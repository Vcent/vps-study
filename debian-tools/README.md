# vps 初始化-v1

1. 以 root 执行./install.sh

```shell
=====================================
          服务器初始化主脚本
=====================================
0. 一键执行所有脚本
1. 创建vv用户（含sudo免密、SSH同步）
2. 禁止 root 用 ssh 登录 && 禁止用 password 登录!!!!!
3. 启用 ssh 的 passwd 登录
4. 安装Docker（含自动启动、测试）
5. 添加512MB Swap分区（永久生效）
6. 部署V2Ray（VLESS+WS+TLS）
7. 部署 mt 服务（Docker 版）
8. 部署本地 subscribe url 服务
9. 部署Xray with vless + xtls + reality 服务
10. 部署fall back 服务
请输入要执行的序号（0-10）：
输入序号：

```

2. 选择需要，对应需要的功能
3. 一般不建议直接选择 0，不建议一键执行所有脚本
4. 一般安装顺序如下：
   1. 创建用户，推荐
   2. 禁止 root 登录，同时禁止 password 登录，需要注意的是，要保持这个 ssh 不退出，否则会把自己永远锁在外面。推荐。
   3. 启用 ssh 的 passwd 登录，一般在本地 publicKey 登录还未配置好的情况使用，推荐。
   4. 安装 docker服务，推荐。
   5. 安装 swap 内存，一般性价比的 vps 都选择 1C1G 配置，内存开启 swap，推荐。
   6. 一键部署 V2Ray 梯子，采用 vless+ws+tls 配置，安全性较高，伪装性良好，配合 cloudflared 的代理隐蔽性非常好。推荐
   7. mt 刮削服务，可选。
   8. 8-10 位梯子的高级选项，不推荐，适合喜欢折腾的高级玩家。
